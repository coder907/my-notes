# Definitions
 - Application: My Notes
 - User: A user of Application



# Purpose

Application facilitates personal note management.



# Features

## Notes management
 - Add a note
 - Edit a note
 - Remove a note
 - Tag a note (with one or more tags)
 - Untag a note

## Notes list
 - Search notes
 - Sort notes

## Tags management
 - Add a tag
 - Edit a tag

## Styles
 - Material
 - Day theme
 - Night theme

## Languages
 - English
 - Slovenian

## Storage
 - Memory
 - Local
 - Cloud

## Navigation
 - Notes management
 - Tags management
 - Settings
 - Log out

## Settings
 - Theme (Day/Night)
 - Language (English/Slovenian)
 - Storage (Memory/Local/Cloud)

  - Hardcoded credentials
